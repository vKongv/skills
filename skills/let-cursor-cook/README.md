# Let Cursor Cook

Orchestrate Cursor CLI agents from Codex, Claude Code, or another external coding harness.

## Install

```bash
npx skills add vkongv/skills --skill let-cursor-cook
```

## Requirements

- Cursor Agent CLI as `agent` or `cursor agent`
- A host harness that can run shell commands

See [SKILL.md](./SKILL.md) for the full skill.
